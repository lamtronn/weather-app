<!-- GETTING STARTED -->
## Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/lamtronn/weather-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run project
   ```js
   npm dev
   ```

   Open http://localhost:3000 with your browser to see the result.
